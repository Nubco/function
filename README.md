# function

Python에서 자주 쓰이는 **핵심 개념과 함수 패턴**을  
`.py 파일 단위`로 정리한 개인 학습·레퍼런스용 저장소입니다.

---

## 📌 목적

이 저장소는 다음을 목표로 합니다.

- Python의 **중요 개념을 코드 중심으로 정리**
- README 설명보다 **`.py 내부 주석과 코드 자체에 집중**
- 알고리즘, 입력 처리, 객체 복사 등 **실전에서 헷갈리기 쉬운 부분을 명확히 구분**
- 나중에 다시 봐도 바로 이해 가능한 **개념 노트 역할**

---

## 📂 구성 방식

- **하나의 파일 = 하나의 개념**
- 모든 설명은 `.py 파일 내부 주석`으로 작성
- README는 전체 구조와 의도만 설명
